# Xcelerator Cache: CCM evidence

CCM diagnostics, cross-checks, validation records, and certificates.

Artifacts and metadata in this repository are generated by the
[Xcelerator Toolkit](https://github.com/TeamXcelerator/xcelerator-toolkit).

## Cache family

**CCM evidence**

CCM diagnostics, cross-checks, validation records, and certificates.

Artifact kinds assigned to this family:

- `ccm_convergence_diagnostics`
- `ccm_cross_check_record`
- `ccm_validation_record`
- `ccm_certificate_bundle`

## Layout

```text
objects/sha256/<first-two-hex>/<full-sha256>
artifacts/<artifact-kind>/<logical-key>/<parameter-digest>/
```

Object payloads are immutable and content-addressed. Artifact manifests provide
logical identity, provenance, validation state, and ordered object references.
The Xcelerator cache registry is the authority for shard routing.
